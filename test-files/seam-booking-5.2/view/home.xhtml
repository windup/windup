About this example application

This sample application demonstrates how easy it is to develop stateful web applications using JBoss Seam. Just register, login, and book a room to see Seam in action. Throughout the application you'll see popup links like the ones at the bottom of this page. Click them to see how the application works under the hood.

Note: Please do NOT enter personal information or your credit card number in this sample application.
